# 🎓 Sphoorthy Event Management System

A full-stack Event Management System developed for **Sphoorthy Engineering College** to streamline the management of clubs and events. The application enables students to explore clubs, register for events, and allows administrators to manage clubs and events through a centralized platform.

## 🚀 Features

- User Authentication (Student & Admin)
- Club Management
- Event Management
- Event Registration
- Role-Based Access Control
- REST API Integration
- MySQL Database

## 🛠️ Tech Stack

**Frontend**
- HTML
- CSS
- JavaScript

**Backend**
- Java
- Spring Boot
- Spring Data JPA
- Maven

**Database**
- MySQL

## ▶️ Getting Started

1. Clone the repository
2. Import the backend into Eclipse/IntelliJ
3. Configure MySQL in `application.properties`
4. Run the Spring Boot application
5. Open the frontend using Live Server
